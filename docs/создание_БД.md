# Шаги
1) установить и настроить PostgreSQL;  
2) правильно заполнить `DATABASE_URL`;  
3) добавить в `webhook_test.py` запись лида в таблицу.

## 1) Установка PostgreSQL

На сервере:

```bash
apt update
apt install -y postgresql postgresql-contrib
systemctl enable postgresql
systemctl start postgresql
systemctl status postgresql --no-pager
```

## 2) Создать БД и пользователя

```bash
sudo -u postgres psql
```

Внутри `psql`:

```sql
CREATE USER wr_wh_leads WITH PASSWORD 'СЮДА_НОВЫЙ_СЛОЖНЫЙ_ПАРОЛЬ';
CREATE DATABASE wr_wh_leads_prod OWNER wr_wh_leads;
GRANT ALL PRIVILEGES ON DATABASE wr_wh_leads_prod TO wr_wh_leads;
\q
```

## 3) Что писать в `.env` (DATABASE_URL)

Пиши так:

```env
DATABASE_URL=postgresql+psycopg://wr_wh_leads:СЮДА_ПАРОЛЬ@127.0.0.1:5432/wr_wh_leads_prod
```

Если в пароле есть спецсимволы (`@`, `:`, `/`, `?`, `#`) — их нужно URL-encode, иначе строка сломается.

## 4) Быстрая проверка подключения

```bash
source /opt/WR_wh_leads/venv/bin/activate
python3 - <<'PY'
import os
from sqlalchemy import create_engine, text
from dotenv import load_dotenv

load_dotenv('/opt/WR_wh_leads/.env')
engine = create_engine(os.getenv('DATABASE_URL'))
with engine.connect() as conn:
    print(conn.execute(text('SELECT 1')).scalar())
PY
```

Должно вывести `1`.
